# Demo Example

A minimal Mesa model used to smoke-test the CI pipeline.

## What it does

Creates a small grid, places a handful of agents, and steps the model
five times. No visualisation — just enough to confirm the core Mesa
API works.

## Running it

```bash
python run.py
```
